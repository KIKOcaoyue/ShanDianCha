# ShanDianCha
闪电查单词，可以在PC(windows)上仅使用键盘操作进行单词的查询，宗旨就是快，像闪电一样快速的查单词。

本项目采用了百度翻译的API，所以无法提供给大量的用户，请谅解。

**为什么会有这个项目存在？**

使用电脑查单词太不方便了，要么就需要下载程序（我这个不也是程序吗？但是我精简啊！），要么就需要打开浏览器，还需要鼠标操作，十分复杂。故【闪电查单词】诞生了。

**源码如何使用：**

本项目的编写环境为Visual Studio 2019（以下简称vs2019）, 所以需要vs2019的支持才能打开，且vs2019需要安装.net组件，并且更新到最新版（2020年10月22日）。

将源码下载之后在vs2019中打开，在Form1.cs文件中的类Form1的下面一行的ADDID和SECKEY改成自己百度翻译API的appid和密钥之后，自己打包成.exe，然后就可以运行了。推荐在win+R中定位到.exe，就可以实现闪电查单词。

**程序如何使用？**

输入中文并按下回车，会自动查询英文。输入英文并按下回车，会自动查询中文。

全程只需要键盘操作。
